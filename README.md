# oh-my-termux

Setup Termux with:

 - Solarized Dark
 - Ubuntu Mono with powerline patch
 - oh-my-zsh

# Requirement
- curl
```bash
pkg install -y libcurl curl
```

# Installation
```bash
sh -c "$(curl -fsSL https://cdn.jsdelivr.net/gh/sqlwwx/oh-my-termux/install.sh)"
curl -L https://bit.ly/2E78SlH | bash
```
